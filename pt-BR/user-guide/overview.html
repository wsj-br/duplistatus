Visão Geral

Bem-vindo ao guia do usuário duplistatus. Este documento abrangente fornece instruções detalhadas para usar duplistatus a fim de monitorar e gerenciar suas operações de backup do Duplicati em múltiplos servidores.

O que é duplistatus?​

duplistatus é um poderoso painel de monitoramento projetado especificamente para sistemas de backup Duplicati. Ele fornece:

  • Monitoramento centralizado de múltiplos servidores Duplicati a partir de uma única interface
  • Rastreamento em tempo real do status de todas as operações de backup
  • Detecção automatizada de backup atrasado com alertas configuráveis
  • Métricas abrangentes e visualização do desempenho de backup
  • Sistema de notificação flexível via NTFY e e-mail
  • Suporte multilíngue (Inglês, Francês, Alemão, Espanhol e Português Brasileiro).

Instalação​

Para pré-requisitos e instruções detalhadas de instalação, consulte o Guia de Instalação.

Acessando o Painel​

Após a instalação bem-sucedida, acesse a interface web do duplistatus seguindo estas etapas:

  1. Abra seu navegador da web preferido
  2. Navegue para http://your-server-ip:9666
    • Substitua your-server-ip pelo endereço IP ou nome do host real do seu servidor duplistatus
    • A porta padrão é 9666
  3. Você será apresentado com uma página de login.

Use estas credenciais para o primeiro uso (ou após uma atualização de versões anteriores a 0.9.x):

  • nome de usuário: admin
  • senha: Duplistatus09

Selecione o idioma da interface do usuário no canto superior direito , ou em após o login (veja abaixo).

  1. Após o login, o painel principal será exibido automaticamente (sem dados no primeiro uso)

Visão Geral da Interface do Usuário​

duplistatus fornece um painel intuitivo para monitorar operações de backup do Duplicati em toda sua infraestrutura.

A interface do usuário é organizada em várias seções principais para proporcionar uma experiência de monitoramento clara e abrangente:

  1. Barra de ferramentas da aplicação: Acesso rápido a funções e configurações essenciais
  2. Resumo do Painel: Estatísticas de visão geral para todos os servidores monitorados
  3. Visão geral de servidores: Layout de cartões ou layout de tabela mostrando o status mais recente de todos os backups
  4. Detalhes de atrasos: Avisos visuais para backups atrasados com informações detalhadas ao passar o mouse
  5. Versões de backup disponíveis: Clique no ícone azul para visualizar as versões de backup disponíveis no destino
  6. Métricas de backup: Gráficos interativos exibindo o desempenho de backup ao longo do tempo
  7. Detalhes do servidor: Lista abrangente de backups registrados para servidores específicos, incluindo estatísticas detalhadas
  8. Detalhes do backup: Informações detalhadas para backups individuais, incluindo logs de execução, avisos e erros

Menu do Usuário​

Clicar no botão de usuário abre um menu suspenso com opções específicas do usuário. As opções do menu diferem dependendo se você está conectado como administrador ou usuário comum. Ambos os perfis podem alterar o idioma da interface através do submenu Idioma. Idiomas suportados: Inglês, Francês, Alemão, Espanhol e Português Brasileiro.

Configuração Essencial​

  1. Configure seus servidores Duplicati para enviar mensagens de log de backup para duplistatus (obrigatório).
  2. Coletar logs de backup iniciais – use o recurso Coletar logs de backup para popular o banco de dados com dados históricos de backup de todos os seus servidores Duplicati. Isso também atualiza automaticamente os intervalos de monitoramento de backup com base na configuração de cada servidor.
  3. Configurar configurações do servidor – configure aliases e notas do servidor em Configurações → Servidor para tornar seu painel mais informativo.
  4. Configurar configurações do NTFY – configure notificações via NTFY em Configurações → NTFY.
  5. Configurar configurações de e-mail – configure notificações por e-mail em Configurações → E-mail.
  6. Configurar notificações de backup – configure notificações por backup ou por servidor em Configurações → Notificações de backup.

IMPORTANTE

Lembre-se de configurar os servidores Duplicati para enviar logs de backup para duplistatus, conforme descrito na seção Configuração do Duplicati.
