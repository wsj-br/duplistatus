# Contributing Translations

Thank you for helping translate duplistatus documentation!

## Process

1. **Join Crowdin project**: [Link to be added after project creation]
2. **Select language**: Choose from French, German, Spanish, or Portuguese (Brazil)
3. **Translate**: Use Crowdin's web interface to translate strings and documentation
4. **Review**: Wait for proofreader approval
5. **Sync**: Translations are automatically merged weekly via GitHub Actions

## Guidelines

- **Keep technical terms consistent** (see glossary below)
- **Don't translate**: Code blocks, URLs, API endpoints, class names, technical terms that should remain English
- **Preserve markdown formatting**: Keep the same structure as the English version
- **Use formal tone**: 
  - French: Use "vous" (formal)
  - German: Use "Sie" (formal)
  - Spanish: Use "usted" (formal)
  - Portuguese: Use "você" (informal) or formal constructions as appropriate
