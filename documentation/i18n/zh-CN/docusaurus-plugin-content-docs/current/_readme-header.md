# duplistatus - 监控多个 [Duplicati](https://github.com/duplicati/duplicati) 服务器

![version](https://img.shields.io/badge/version-{VERSION}-blue) 
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?logo=github)](https://github.com/wsj-br/duplistatus)

<br/>

该 Web 应用程序用于监控和可视化 [Duplicati](https://github.com/duplicati/duplicati) 的备份操作。**duplistatus** 提供了一个全面的仪表板，用于跟踪多个服务器的备份状态、通知逾期备份、查看作业历史、指标和性能。

可通过 Docker、Podman 或 Portainer 轻松部署，并提供 6 种语言（英语、德语、法语、西班牙语、葡萄牙语和简体中文），每种语言均配有完整的文档。
