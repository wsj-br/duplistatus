# 环境变量 {#environment-variables}

该应用程序支持以下用于配置的环境变量：

| 变量                  | 描述                                                                                 | 默认                    |
|---------------------------|---------------------------------------------------------------------------------------------|:---------------------------|
| `PORT`                    | 主 Web 应用程序的端口                                                           | `9666`                     |
| `CRON_PORT`               | Cron 服务（调度）的端口。如果未设置，则使用 `PORT + 1`                                      | `9667`                     |
| `NODE_ENV`                | Node.js 环境 (`development` 或 `production`)                                         | `production`               |
| `NEXT_TELEMETRY_DISABLED` | 禁用 Next.js 遥测                                                                   | `1`                        |
| `TZ`                      | 应用程序的时区                                                                | `Europe/London`            |
| `PWD_ENFORCE`             | 设置为 `false` 以禁用密码复杂度要求（大写字母、小写字母、数字）。 | 强制执行（完整验证） |
| `PWD_MIN_LEN`             | 密码的最小字符长度（始终强制执行）                                    | `8`                        |
