

# 环境变量 {#environment-variables}

应用程序支持以下环境变量进行配置：

| 变量                  | 说明                                                                                 | 默认值                    |
|---------------------------|---------------------------------------------------------------------------------------------|:---------------------------|
| `PORT`                    | 主 Web 应用程序端口                                                           | `9666`                     |
| `CRON_PORT`               | Cron 服务（调度）端口。若未设置，使用 `PORT + 1`                                      | `9667`                     |
| `NODE_ENV`                | Node.js 环境（`development` 或 `production`）                                         | `production`               |
| `NEXT_TELEMETRY_DISABLED` | 禁用 Next.js 遥测                                                                   | `1`                        |
| `TZ`                      | 应用程序时区                                                                | `Europe/London`            |
| `PWD_ENFORCE`             | 设为 `false` 可禁用密码复杂度要求（大写、小写、数字）。 | 强制（完整验证） |
| `PWD_MIN_LEN`             | 最小密码长度（字符数）（始终强制）                                    | `8`                        |
