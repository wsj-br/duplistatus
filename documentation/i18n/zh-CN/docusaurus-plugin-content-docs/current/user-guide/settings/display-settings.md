# 显示 {#display}

配置用户界面和显示首选项。

![Display Settings](../../assets/screen-settings-display.png)

<br/>

| 设置                   | 描述                                               | 默认值      |
|:--------------------------|:----------------------------------------------------|:-------------------|
| **Table Size**            | 服务器详情页每页显示的行数。 | 5 行             |
| **Theme**                 | 选择浅色、深色或匹配您的操作系统外观（首选浅色/深色模式）。 | 未设置时遵循操作系统 |
| **Chart Time Range**      | 图表中显示的时间间隔。可用选项：**1W**（上次 7 天）、**2W**（上次 14 天）、**1M**（上次 30 天）、**3M**（上次 90 天）。您也可以直接从图表标题中切换时间范围。 | 1 个月            |
| **Chart Style**           | 在平滑曲线图或柱状图可视化之间进行选择。两种模式均使用时间桶聚合以实现最佳显示。您也可以直接从图表标题中切换。 | 平滑曲线       |
| **Format Locale**         | 选择一个独立于 UI 语言的格式区域设置（支持 416 个区域设置）。这将影响日期、时间和数字的显示方式。选择时会显示实时预览。示例：UI 语言 = 德语，格式区域设置 = 英语 (英国) $ightarrow$ 德语 UI 配合英国日期格式。 | 基于 UI 语言 |
| **Auto-refresh Interval** | 页面自动刷新的频率。              | 1 分钟           |
| **Cards Sort Order**      | 仪表板上卡片的排序方式。              | 服务器名称 (a-z)  |
| **Start of Week**         | 配置一周起始日。                     | 基于区域设置    |

<br/>

:::tip
**快速访问**：您可以通过右键单击应用程序工具栏中的自动刷新按钮快速访问此页面。
:::
