# duplistatus - 监控多个 [Duplicati](https://github.com/duplicati/duplicati) 服务器

![版本](https://img.shields.io/badge/version-{VERSION}-blue) 
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?logo=github)](https://github.com/wsj-br/duplistatus)

<br/>

此 Web 应用程序监控和可视化 [Duplicati](https://github.com/duplicati/duplicati) 的备份操作。 **duplistatus** 提供了一个全面的仪表板来跟踪备份状态，通知逾期备份，作业历史，指标和多个服务器的性能。

易于使用 Docker、Podman 或 Portainer 进行部署，提供 7 种语言（英语、德语、法语、西班牙语、葡萄牙语、印地语（罗马化）和简体中文），且每种语言均包含完整文档。
