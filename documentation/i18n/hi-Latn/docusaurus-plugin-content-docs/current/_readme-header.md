# duplistatus - Ekadhik [Duplicati's](https://github.com/duplicati/duplicati) Serveron ka Monitor Karen

![version](https://img.shields.io/badge/version-{VERSION}-blue) 
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?logo=github)](https://github.com/wsj-br/duplistatus)

<br/>

Yah web application [Duplicati](https://github.com/duplicati/duplicati) se backup operations ko monitor aur visualize karti hai. **duplistatus** backup statuses, overdue backups ki notification, job history,
metrics, aur multiple servers par performance ko track karne ke liye ek comprehensive dashboard pradaan karta hai.

Aasani se deploy karne ke liye Docker, Podman, ya Portainer ka upyog kiya ja sakta hai aur 7 Bhaashao mein uplabdh hai (Angrezi, German, French, Spanish, Portuguese, Hindi (Roman), aur Simplified Chinese), aur pratyek Bhaasha mein poora documentation hai.
