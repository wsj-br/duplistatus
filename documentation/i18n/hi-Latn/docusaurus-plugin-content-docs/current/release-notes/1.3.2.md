# Sanskaran 1.3.2 {#version-132}

## Overview {#overview}

Sanskaran 1.3.2 ek maintenance release hai jo dependencies mein security vulnerabilities ko address karta hai, pnpm install warnings (peer webpack aur deprecated subdependencies) ko fix karta hai, app aur documentation mein lint aur build warnings ko resolve karta hai, aur vartaman Docker script names aur package versions ko reflect karne ke liye documentation ko update karta hai.

## Suraksha {#security}

### Dependency vulnerability fixes {#dependency-vulnerability-fixes}

- **Dependabot dependency updates (documentation)**: Documentation workspace mein vulnerabilities ko address karne ke liye do Dependabot PRs merge kiye gaye:
  - **ajv** `/documentation` mein: 8.17.1 se 8.18.0 aur 6.12.6 se 6.14.0 tak update kiya gaya.
  - **fast-xml-parser** `/documentation` mein: 5.3.5 se 5.3.8 tak update kiya gaya.
  Dono ko /documentation ke liye dependabot multi-update ke through merge kiya gaya.

- **High-severity vulnerabilities ke liye pnpm overrides**: Char high-severity issues ko fix karne ke liye pnpm overrides add kiye gaye:
  - **minimatch** (>=3.1.4): documentation > @docusaurus/core > serve-handler > minimatch mein GHSA-3ppc-4f35-3m26 (repeated wildcards ke through ReDoS), GHSA-7r86-cg39-jmmj (combinatorial backtracking), aur GHSA-23c5-xmqv-rm74 (nested extglobs backtracking) ko fix karta hai.
  - **serialize-javascript** (>=7.0.3): webpack > terser-webpack-plugin > serialize-javascript mein GHSA-5c6j-r48x-rmvq (RegExp.flags aur Date.prototype.toISOString() ke through RCE) ko fix karta hai.

## Bug fixes {#bug-fixes}

- **pnpm install chetaavaniyaan**: Teen install-samay ki chetaavaniyaan hal kien: (1) `webpack@5.104.1` se `next-intlayer` > `@intlayer/webpack` (project mein webpack 5.105.3 hai) se unmet peer ko root `package.json` mein `pnpm.peerDependencyRules.allowedVersions.webpack: "5"` jodkar hal kiya. (2) Deprecated subdependencies `prebuild-install@7.1.3` (better-sqlite3 se) aur `whatwg-encoding@3.1.1` (documentation > docusaurus-search-local > cheerio se) ko `pnpm.allowedDeprecatedVersions` jodkar hal kiya taaki install chetaavaniyon ke bina chale.
- **Application logs viewer (exhaustive-deps)**: Auto-scroll effect ke liye react-hooks/exhaustive-deps chetaavani hal ki. `logData` ko intentional taur par dependency array se chhod diya gaya hai taaki scrolling kewal tab ho jab auto-scroll ya selected file badle, har poll par nahin; new-line scrolling `loadLogs` mein handle kiya gaya hai. Ek eslint-disable-next-line ek chhote comment ke saath joda gaya hai.
- **Build warning vscode-languageserver-types (documentation)**: Webpack "Critical dependency: require function is used in a way in which dependencies cannot be statically extracted" ko transitive dependency `vscode-languageserver-types` (intlayer-editor se) ke liye documentation (Docusaurus) build mein suppress kiya gaya hai, `documentation/docusaurus.config.ts` mein ek inline plugin ke dwara jo `ignoreWarnings` jodta hai. Chetaavani kewal docs build ke dauraan hoti hai, main app ke nahin.

## Migration notes {#migration-notes}

### From version 1.3.1 {#from-version-131}

Version 1.3.2 mein upgrade karte samay:

1. **No database migration required**: Is release ke liye kisi database schema change ki zaroorat nahi hai.
2. **Docker commands**: Agar aap documented pnpm scripts use karte hain, to pichle `pnpm docker:up`, `pnpm docker:down`, ityadi ke bajaye `pnpm docker:clean`, `pnpm docker:devel`, `docker-up`, aur `docker-down` use karein.
3. **Dependencies**: Updated lockfile aur overrides ko apply karne ke liye `pnpm install` run karein; typical installations ke liye kisi manual dependency change ki zaroorat nahi hai.
