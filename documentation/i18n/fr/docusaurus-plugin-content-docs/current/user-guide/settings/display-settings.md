# Affichage {#display}

Configurer les préférences d'interface utilisateur et d'affichage.

![Display Settings](../../assets/screen-settings-display.png)

<br/>

| Paramètre                   | Description                                         | Valeur par défaut      |
|:----------------------------|:----------------------------------------------------|:-----------------------|
| **Taille du tableau**       | Nombre de lignes par page sur la page des détails du serveur. | 5 lignes             |
| **Thème**                 | Choisissez clair, sombre ou adaptez-vous à l'apparence de votre système d'exploitation (mode clair/sombre préféré). | Suivre le système d'exploitation par défaut |
| **Plage de temps du graphique**      | Intervalle de temps affiché dans les graphiques. Options disponibles : **1W** (7 derniers jours), **2W** (14 derniers jours), **1M** (30 derniers jours), **3M** (90 derniers jours). Vous pouvez également basculer la plage de temps directement depuis les en-têtes des graphiques. | 1 mois            |
| **Style de graphique**           | Choisissez entre des graphiques en lignes lissées ou en barres. Les deux modes utilisent une agrégation par intervalles temporels pour un affichage optimal. Vous pouvez également basculer directement depuis les en-têtes des graphiques. | Lignes lissées       |
| **Paramètres régionaux**         | Sélectionnez des paramètres régionaux de formatage indépendants de la langue de l'interface (416 paramètres régionaux pris en charge). Cela affecte l'affichage des dates, heures et nombres. Un aperçu en direct est affiché lors de votre sélection. Exemple : Langue de l'interface = allemand, Paramètres régionaux de formatage = anglais (Royaume-Uni) → interface en allemand avec formats de date du Royaume-Uni. | Basé sur la langue de l'interface |
| **Intervalle de rafraîchissement automatique** | Fréquence de rafraîchissement automatique des pages.              | 1 minute           |
| **Ordre de tri des cartes**      | Ordre de tri des cartes sur le tableau de bord.              | Nom du serveur (a-z)  |
| **Début de la semaine**         | Configurez à quel moment la semaine commence.                     | Basé sur les paramètres régionaux    |

<br/>

:::tip
**Accès rapide** : Vous pouvez accéder rapidement à cette page en cliquant avec le bouton droit sur le bouton d'actualisation automatique dans la barre d'outils de l'application.
:::
