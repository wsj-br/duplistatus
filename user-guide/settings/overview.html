Overview

The Settings page offers a unified interface for configuring all aspects of duplistatus. You can access it by clicking the Settings button in the Application Toolbar. Note that regular users will see a simplified menu with fewer options compared to administrators.

Administrator View​

Administrators see all available settings.

  • Notifications
    • Backup Notifications: Configure per-backup notification settings
    • Backup Monitoring: Configure overdue backup detection and alerts
    • Templates: Customise notification message templates

  • Integrations
    • NTFY: Configure NTFY push notification service
    • Email: Configure SMTP email notifications

  • System
    • Servers: Manage Duplicati server configurations
    • Display Settings: Configure theme, chart time range, chart style, format locale, auto-refresh interval, card sort order, and week start
    • Database Maintenance: Perform database cleanup (admin only)
    • Users: Manage user accounts (admin only)
    • Audit Log: View system audit logs
    • Audit Log Retention: Configure audit log retention (admin only)
    • Application Logs: View and export application logs (admin only)

Non-Administrator View​

Regular users see a limited set of settings.

  • Notifications
    • Backup Notifications: View per-backup notification settings (read-only)
    • Backup monitoring: View overdue backup settings (read-only)
    • Templates: View notification templates (read-only)

  • Integrations
    • NTFY: View NTFY settings (read-only)
    • Email: View email settings (read-only)

  • System
    • Servers: View server configurations (read-only)
    • Display: Configure theme, chart time range, chart style, format locale, auto-refresh interval, card sort order, and week start
    • Audit Log: View system audit logs (read-only)

Status Icons​

The sidebar displays status icons next to the NTFY and Email integration settings:

  • Green icon: Your settings are valid and configured correctly
  • Yellow icon: Your settings are not valid or not configured

When the configuration is invalid, the corresponding checkboxes in the Backup Notifications tab will be greyed out and disabled. For more details, see the NTFY Settings and Email Settings pages.


important

A green icon does not necessarily mean that notifications are functioning correctly. Always use the available test features to confirm your notifications are working before relying on them.
