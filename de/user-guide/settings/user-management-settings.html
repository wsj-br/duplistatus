Benutzer

Verwalten Sie Benutzerkonten, Berechtigungen und Zugriffskontrolle für duplistatus. Dieser Abschnitt ermöglicht Administratoren das Erstellen, Ändern und Löschen von Benutzerkonten.

Tip

Das Standard-Konto admin kann gelöscht werden. Erstellen Sie dazu zunächst einen neuen Administratorbenutzer, melden Sie sich mit diesem Konto an, und löschen Sie dann das Konto admin.

Das Standard-Passwort für das Konto admin ist Duplistatus09. Sie müssen es beim ersten Anmelden ändern.

Benutzerverwaltung​

Sie können auf den Bereich Benutzerverwaltung auf zwei Arten zugreifen:

  1. Aus dem Benutzermenü: Klicken Sie auf in der Anwendungssymbolleiste und wählen Sie "Admin-Benutzer" aus.

  2. Aus Einstellungen: Klicken Sie auf und Benutzer in der Einstellungen-Seitenleiste

Erstellen eines neuen Benutzers​

  1. Klicken Sie auf die Schaltfläche
  2. Geben Sie die Benutzerdetails ein:
    • Benutzername: Muss 3–50 Zeichen lang sein, eindeutig und ohne Beachtung der Groß-/Kleinschreibung
    • Administrator: Aktivieren Sie das Kontrollkästchen, um Administratorrechte zu vergeben
    • Passwortänderung erforderlich: Aktivieren Sie das Kontrollkästchen, um die Passwortänderung beim ersten Anmelden zu erzwingen
    • Passwort:
      • Option 1: Aktivieren Sie "Passwort automatisch generieren", um ein sicheres temporäres Passwort zu erstellen
      • Option 2: Deaktivieren Sie die Option und geben Sie ein benutzerdefiniertes Passwort ein
  3. Klicken Sie auf .

Benutzer bearbeiten​

  1. Klicken Sie auf das -Bearbeitungssymbol neben dem Benutzer
  2. Ändern Sie eine der folgenden Angaben:
    • Benutzername: Ändern Sie den Benutzernamen (muss eindeutig sein)
    • Administrator: Aktivieren oder deaktivieren Sie Administratorrechte
    • Passwortänderung erforderlich: Aktivieren oder deaktivieren Sie die Anforderung zur Passwortänderung
  3. Klicken Sie auf .

Passwort eines Benutzers zurücksetzen​

  1. Klicken Sie auf das Schlüsselsymbol neben dem Benutzer
  2. Bestätigen Sie das Passwort zurücksetzen
  3. Ein neues temporäres Passwort wird generiert und angezeigt
  4. Kopieren Sie das Passwort und geben Sie es dem Benutzer sicher weiter

Benutzer löschen​

  1. Klicken Sie auf das Löschsymbol neben dem Benutzer
  2. Bestätigen Sie das Löschen im Dialogfeld. Das Löschen von Benutzern ist dauerhaft und kann nicht rückgängig gemacht werden.

Kontosperrung​

Konten werden nach mehreren fehlgeschlagenen Anmeldeversuchen automatisch gesperrt:

  • Sperrungsschwelle: 5 fehlgeschlagene Versuche
  • Sperrdauer: 15 Minuten
  • Gesperrte Konten können sich nicht anmelden, bis die Sperrfrist abläuft

Wiederherstellung des Admin-Zugriffs​

Wenn Sie Ihr Admin-Passwort verloren haben oder aus Ihrem Konto gesperrt wurden, können Sie den Zugriff mithilfe des Admin-Wiederherstellungsskripts wiederherstellen. Weitere Informationen finden Sie im Leitfaden Admin Account Recovery mit detaillierten Anweisungen zur Wiederherstellung von Administrator-Zugriff in Docker-Umgebungen.
