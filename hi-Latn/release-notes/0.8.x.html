Sanskaran 0.8.x

Overview​

Sanskaran 0.8.x ek mahatvapurna update hai jo mahatvapurna suraksha badhotari, behtar server prabandhan kshamataayein, aur SMTP email support ke saath ek vyapak suchana pranali pesh karta hai. Yah release suraksha ko majboot karne, upyogkarta anubhav ko badhane, aur adhik flexible backup monitoring vikalp pradaan karne par kendrit hai.

New Features​

Server Prabandhan Badhotari​

  • Automatic server URL aur password persistence backup collect karte samay
    • Upyogkarta URL aur password do tareekon se update kar sakte hain:
      • Updated values ke saath backup dobara collect karna
      • Sammaan → Server Sammaan mein settings modify karna

Behtar Backup Monitoring​

  • Naya Backup monitoring configuration tab Duplicati server-compatible interval settings ke saath
    • Custom intervals ke liye support (jaise, "1D12h")
    • Backup log collection ke dauraan Duplicati configuration se automatic overdue interval updates
    • Sifarish ki jaati hai: Duplicati server mein backup job intervals badalne ke baad collection run karein taaki duplistatus configuration sync ho sake
    • Suchana templates ab {backup_interval} ka istemaal karte hain {backup_interval_value} aur {backup_interval_type} ke bajaye.

NTFY Device Configuration​

  • QR code generation automatic device configuration ke liye duplistatus se suchnaayein prapt karne ke liye
    • Application toolbar mein View NTFY Messages button par right-click karein
    • Sammaan → NTFY Sammaan mein "Configure Device" button

Behtar Backup Collection​

  • Automatic HTTP/HTTPS detection, connections is kram mein try kiye jaate hain: HTTPS → HTTPS with self-signed → HTTP fallback
  • Collect Backup Logs interface mein Server selection dropdown seedhe server-specific collection ke liye
  • Hostname mein multiple addresses (comma-separated) ab sweekar kiye jaate hain (same password aur port share karna chahiye)
  • Dashboard (table), Sammaan → Backup Monitoring aur Sammaan → Server mein Single-server backup collection buttons jode gaye hain
  • Bulk collection functionality sabhi servers ke liye valid configuration (url aur password) ke saath
    • Sammaan → Backup Monitoring aur Sammaan → Server mein Collect All buttons
    • Application toolbar mein Collect Backup Logs button par right-click context menu

Behtar Suchana Pranali​

  • SMTP Email Suchana Support:

    • Sammaan → Email Sammaan mein SMTP server configuration
    • NTFY aur/ya email suchnaayein ke liye prati-backup job configuration
    • HTML-formatted templates jo maujooda Sammaan → Suchana Templates ka istemaal karte hain

  • Prati-Backup Job Suchana Configuration:

    • Sammaan → Backup Suchnaayein mein vyaktigat suchana pasandeeda
    • Visual indicators (greyed icons) jab NTFY ya email sahi se configure na ho

  • Locale support:

    • application mein locale set karne ke liye naya environment variable LANG banaya gaya hai
    • ab notifications aur logs mein dates aur numbers ka format anumanit hai

🔒 Suraksha Badhotari​

CSRF Protection​

  • Session-based authentication majboot session prabandhan ke saath
  • Sabhi state-changing API requests ke liye CSRF token validation lagu kiya gaya hai
  • Sessions automatically expire hoti hain 24 ghante baad; CSRF tokens har 30 minute mein refresh hote hain
  • Cross-Site Request Forgery ke khilaaf suraksha sunishchit karta hai jabki external APIs ke saath compatibility bani rehti hai

Sensitive Data ke liye Plaintext Password Minimization​

  • Passwords sirf user interface ke through set kiye ja sakte hain; koi API endpoint stored passwords ko expose nahi karta hai
  • Plaintext password manipulation poore system mein minimize kiya gaya hai

Sensitive Data ke liye Advanced Cryptography​

  • Sanvedansheel data encryption (jaise, passwords, SMTP credentials) AES-256-GCM ka upyog karke
  • Master key svachaalit roop se banayi jaati hai aur .duplistatus.key mein surakshit roop se store ki jaati hai
  • 100,000 iterations ke saath PBKDF2 suraksha ko mazboot karne ke liye key derivation ke liye upyog kiya jaata hai
  • Authentication tags ki jaanch ki jaati hai aur istemaal ke baad memory ko surakshit roop se clear kar diya jaata hai
  • Master key file permissions ko adhiktam suraksha ke liye sakhti se 0400 par set kiya gaya hai

🎨 Upyogkarta Interface Sudhaar​

  • Behtar upyogita ke liye naye rang scheme aur iconography ke saath enhanced application styling
  • Sudhare hue status indicators ke saath aadhunik design system
  • Behtar drishya pratikriya ke liye rangon wale icons aur progress indicators
  • Upyogita ko suvidhajanak banane ke liye Settings tabs ka naam badla gaya hai
  • Streamlined API interactions aur behtar upyogkarta anubhav

📚 Dastaavej Sudhaar​

  • Behtar navigation aur khoj ke liye Docusaurus ka upyog karke comprehensive documentation
  • Vistrit endpoint varnanon ke saath enhanced API documentation
  • Nayi suvidhaon aur suraksha sudhaaron ko cover karne wale updated user guides
  • Behtar setup instructions ke saath installation guides
  • Yogdaankartaon ke liye Development documentation

🔧 Takniki Sudhaar​

  • Behtar performance aur upyogkarta anubhav ke saath enhanced Next.js configuration
  • Preload warnings ko kam karne ke liye optimised bundle splitting
  • Connection issues aur API interactions ke liye behtar error handling
  • Behtar configuration handling ke liye backup settings management ko refactor kiya gaya hai
  • API interactions mein enhanced session management aur CSRF protection
  • Behtar database migrations aur schema updates
  • Kayi bug fixes aur code simplification.

🚀 Migration Notes​

Sanskaran 0.7.x se​

Is release mein mahatvapurna suraksha sudhaar aur nayi suvidhaen shamil hain. Sanskaran 0.7.x se upgrade karte samay:

  1. Automatic Database Migration: Nayi suvidhaon ka samarthan karne ke liye application svachaalit roop se aapke database schema ko migrate karegi
  2. Master Key Generation: Sanvedansheel data ki encryption ke liye ek nayi master key banayi jayegi
  3. Session Management: Maujooda sessions ko avaidh kar diya jayega aur nayi CSRF-protected sessions sthapit ki jayengi
  4. Configuration Updates: Nayi suvidhaon ka samarthan karne ke liye kuchh configuration keys ko update kiya gaya hai
  5. Template changes: {backup_interval_value} aur {backup_interval_type} variables ko {backup_interval} se badal diya gaya hai. Default templates svachaalit roop se adjust ho jayenge, lekin custom templates migrate nahi honge. Kripya apne templates ki jaanch karein.

Suraksha Sambandhi Vichar​

  • Master Key File: .duplistatus.key file ko uchit roop se backup aur surakshit kiya gaya hai, yah sunishchit karein
  • File Permissions: Suraksha ke liye master key file mein pratibandhit permissions (0400) hongi
  • Password Encryption: Maujooda passwords ko naye cryptographic system ka upyog karke encrypt kiya jayega

🐛 Bug Fixes​

  • 15 sekand se kam samay ke antar ke liye "abhi" return karne ke liye relative time formatting ko theek kiya gaya
  • Missing environment variables ke liye error handling ko behtar banaya gaya
  • Enhanced connection testing aur validation
  • Vibhinn UI inconsistencies ko theek kiya gaya aur upyogkarta pratikriya ko behtar banaya gaya

Yah release suraksha, upyogita, aur karyakshamata mein ek mahatvapurna kadam hai. Ham naye features aur suraksha sudharon ka labh uthane ke liye upgrade karne ki sifarish karte hain.
