Sanskaran 1.4.2

Overview​

Sanskaran 1.4.2 ek maintenance release hai jo vyaapak bhaasha samarthan, vishwasaniyata, aur dependency hygiene par kendrit hai. Ismein ek poori tarah se localized Simplified Chinese aur Hindi (Roman) antarmukh prastut kiya gaya hai, dakyumenteshan ko saat bhaashayon mein vistaar kiya gaya hai, aur ek mahatvapurn Docker startup crash, saath hi kuch dashboard aur server merge issues ka samadhan kiya gaya hai. Nirbharataen bhi update ki gayi hain, jismein nodemailer ka ek mukhya upgrade shamil hai.

Naye features​

Bhaasha samarthan​

  • Simplified Chinese aur Hindi (Roman) antarmukh: Aplikeshan antarmukh ab Simplified Chinese aur Hindi (Roman) mein poori tarah se uplabdh hai, jo English, French, German, Spanish, aur Brazilian Portuguese mein shamil ho gaya hai. Sammaan mein bhaasha menu se unhein chunein.
  • Vistaarit dakyumenteshan bhaashayen: Online dakyumenteshan ab saat bhaashayon mein prakaashit hoti hai, jo maujood anuvadon ke saath Hindi (Roman) aur Simplified Chinese jodti hai.

Sudhaar​

Backup collection​

  • Stabl server identity kab log collect karte hai: Kab backup log collect karte hai, duplistatus ab Duplicati machine name ko tumhare existing server se match karta hai aur reuse karta hai us server ka record, jaise ki Duplicati ka machine ID par depend karne ke bajaye. Iska matlab hai ki backup itihas ko same server se attach rakhte hai, jaise ki Duplicati ka machine ID badal jaye (for example, after a reinstall), jiske karan duplicate server entries kam hote hai.

Dastaavez​

  • Naya bhaasha suchi: README aur dastavej ab sabhi saat samarthit bhaashaon ka varnan karte hain aur pratyek anuvaadit sanskaran ka link dete hain.
  • Dastavej bhaasha menu jhanda: Naye Hindi aur Simplified Chinese dastavej ke liye jhanda icons jode gaye hain, aur Portuguese ko ekta ke liye Portugal jhande par switch kiya gaya hai. Jhande ab Windows, Linux, aur Chromium par bina kisi external font downloads ke vishwasniya roop se render hote hain.
  • Duplicate-servers margdarshan ka vistaar: Upyogkarta guide ab duplicated servers ke dono kaaranon ka varnan karti hai (badla hua Duplicati machine ID, aur ek Duplicati API bug jo identity ID aur machine ID ko milata hai) aur Duplicati taraf se workaround ko duplistatus merge kadam ke saath dastavej karti hai.

Dashboard performance​

  • Load par kam sign-in jaanchen: Dashboard ab aapka vartaman upyogkarta ek baar lata hai aur ise poore prushth (header, settings guard, vastu shaili aur widgets) mein baanta hai, balki har component alag se maangne ke bajaye, jab prushth load hota hai to faltu anurodhon ko kam karta hai.

Bug fixes​

  • Docker startup crash: Container shuru hote waqt ek crash ko theek kiya jo Cannot find module '.../@swc/helpers/...' report karta tha, jo shayad aplikeshan ke shuru hone se rok sakta tha. Containers ab niyamit roop se shuru hote hain.
  • Duplicate dashboard praveshtiyan: Dashboard ab usi backup ke liye ek se adhik patriyon nahi dikhata jab kayi rekord same latest Backup Taareekh ko baante hain.
  • Duplicate servers ka vilayan: Servers ka vilayan ab duplicate Duplicati backup IDs ko bhi jodta hai jo same Backup Naam baante hain, aur sabse naye backup se ID rakhta hai. Vilayan kriyaayan ab Audit log mein darj hoti hain.

Suraksha​

Dependency updates​

  • nodemailer ko 9.x mein upgrade kiya gaya: Email library ko nayi major sanskaran mein update kiya gaya. v9 mein ekmatra breaking change (remote attachment, OAuth2, ya proxy content lane waqt kathor TLS valideshan) duplistatus ko prabhavit nahi karta, jo sirf inline text aur HTML email bhejta hai.
  • Anya dependency updates: Kai build aur type dependencies ko unki nayi anukool sanskaranon mein refresh kiya gaya.

Migration notes​

Sanskaran 1.4.1 se​

Sanskaran 1.4.2 mein upgrade karte samay:

  1. Koi database migration aavashyak nahi: Is release mein koi database schema parivarton ki zaroorat nahi hai.
  2. Dependencies: Update kiye gaye lockfile aur overrides lagane ke liye pnpm install chalayen.
  3. Koi configuration parivarton ki zaroorat nahi: Vartamaan sammaan, jisme aapki chuni gayi bhaasha shamil hai, surakshit hain.

Changelog​

Sanskaran 1.4.2 mein badlav​

  • Jodha: Poora Simplifaiyd Chini aur Hindi (Roman) upyogkarta anveshan.
  • Jodha: Abaa dokumantashan saat bhaashao me uplabdh hai, jisme Hindi (Roman) aur Simplifaiyd Chini shamil hai.
  • Badlaava: Bakap log sangrah aba Duplicati mashin naam se server ko pahachaanata hai (mauzud rakhne wale record ko phir se upyog karta hai), jisse ki itihas bhi machine ID badalne par bhi lagaa rakhaya jaata hai.
  • Badlaava: Dokumantashan bhaasha menu me naye sthaan ke liye jhande jodhe gaye hai aur Portugees ke liye Portugal ka jhanda upyog kiya gaya hai.
  • Badal diya gaya: Doksumenteshan aur README ko saare samarthit bhashaon ka vivran karne aur duura-server samasya-nivaaran maargdarshan ko vishal karne ke liye aapdat kiya gaya hai.
  • Badal diya gaya: nodemailer ko 9.x tak upgrade kiya aur dusre banane/prakar aavashyaktaon ko taaza kiya.
  • Sudhaar diya gaya: Docker shuru karne ke samay ek ghumit @swc/helpers modyul ke abhaav ke karan.
  • Sudhaar diya gaya: Ek se jyada dashboard pravesh jab kee baar kee baackap kee sabse aakhiri taareekh ek hi ho.
  • Thik kiya gaya: Duplicate servers ko maerga karne se ab duplicate backup IDs bhi consolidate ho jaatay hain aur audit log mein record hota hai.
  • Thik kiya gaya: Dashboard load karne ke waqt redundant sign-in checks ko kam kiya gaya hai.
